# Changelog

## 1.0.0 — Initial Release

### Features
- Infinite music mix from liked songs, artists, moods, and playlists
- 12 mood presets (Chill, Focus, Hype, Sad, Drive, Romantic, Party, Workout, Sleep, Acoustic, Indie, Electronic)
- Now Playing card with Like, Dislike, More like this, Discovery Only, Mix from Track, Save as Playlist, Share
- Smart auto-reseed every 5–8 tracks
- Skip streak detection (3 skips → auto reseed)
- Artist learning model (likes +4, dislikes -10)
- Track blacklist (disliked tracks never return)
- Pin system for moods, artists, and playlists
- Home banner widget with quick-launch buttons
- History tab (last 20 tracks)
- Stats tab (session stats, top/bottom artists)
- Share links — copy a URL that recreates your mix
- Save mix as Spotify playlist
- Bilingual onboarding guide (EN/RU)
- Performance optimization — all animations pause when Spotify is hidden
- One-click installer (`install.bat`) and uninstaller (`uninstall.bat`)
